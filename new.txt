下载git

使用git输入你的邮箱和用户名

git config --global user.email "..."

创建文件夹

进入文件夹

初始化文件夹变为git仓库

git init

初始化

git add 

添加动作

git commit -m "说明" 

托管到仓库

git status 

查看仓库当前状态

git diff

查看修改内容

如果文件修改了就需要重新添加使用
git add  
git commit -m "....."

git log 

查看历史版本记录

$ git log --pretty=oneline

查看历史记录基础简单版只显示commit的ID

$ git reset --hard "commit 的 ID"
回滚到ID的版本

git reflog
会打印你的每一次记录,如果想回滚到历史版本没有了ID,可以使用这个命令
